Import Functions

The dataset that will be used is the compressed file that is located in the Info folder. Navigate to Functions>Info folder and decompress the file. After decompressing open MATLAB and and navigate to the location on your computer where you downloaded PupilPlot to. Type at the command promt: >> start_PupilPlotGUI This will open the PupilPlot: Image1 From the File Menu select Load > New files This will open a select file window. Navigate to the location where you decompressed the files and select all of the .txt files in the folder. Image 2 After selecting all the .txt files and pressing Open, a new select Directory window will open. This enables the user to either create a new folder to save the imported files in or select an existing folder to save the imported files in. In this instance we will make a new folder to store the imported files in. Image 3 The results of the import functions will be displayed in the MATLAB command window and the Graph: text will change to Files Loaded. The successful importation of eye tracking files will result in an equal number of files that have the same name as the original files, but will have been converted into .mat files and will also have _IMP appended to the end. E.g.: Subject_1_demoData.txt -------> Subject_1_demoData_IMP.mat This completes the import functions. Return to Step by Step Walk Through or PupilPlot
