Import PreProc Options

Using this Preprocess Menu option allows you to import already defined preprocessing options that were either defined for the current data set or another. After clicking this option a select file window will open in the current working directory and only show files that end with "_preproc.mat", which is the name given to the preprocessing options file generated by PupilPlot. By selecting a preprocessing file the options will be loaded and imported into the current preprocessing options. These can then be either examined by opening the Set PreProc Options window or by opening the log: File Menu > View Log.
