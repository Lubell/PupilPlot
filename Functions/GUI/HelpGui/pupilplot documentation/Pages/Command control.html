Command control

This button group contains the buttons that process the actions associated with their function group.
  • Load
    • pressing this button runs the Load commands
  • Check Values
    • checks the preprocess settings to make sure that everything is okay i.e. nothing missing or crazy
  • Preprocess
    • Runs the preprocessing functions using the options set in the preprocess window
  • Run Analysis
    • Runs analysis functions that takes the preprocessed files and runs the analysis necessary for plotting and stats
