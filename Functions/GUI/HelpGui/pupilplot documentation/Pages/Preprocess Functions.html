Preprocess Functions

After successfully completing the Import Functions the newly imported files are loaded in PupilPlot and ready to be preprocessed. In the demo dataset there were many triggers (also called annotations, henceforth annotations will be referred to as triggers) inserted into the data file while the data was being collected. The only triggers that are of interest for the purposes of this example are:
  • ControlC
  • GlareC
  • GlareP
  • ControlP
and the details of the experiment were such that the actual period of interest was 3500 milliseconds long and that the baseline period was 1000 milliseconds long. The sampling rate of the eye tracker was set at 60 Hz. First the preprocessing options must be set. Go to Preprocess Menu > Set PreProc Options this will open a new window. What the different settings or parameters mean and how to change them is detailed in Set PreProc Options section. For the purposes of this walk through we are only interested in setting that parameters to reflect the experimental details. Here it can be seen that the default smoothing parameter of 10 is being used, that the epoch length has been entered as 3500 ms, the baseline period of 1000 ms has been entered, that the trial length is 4500 ms, triggers are located at the end of the baseline period has been selected, and the sample rate is 60 Hz. preProcGuiSetOptPP_Example3.png By pressing the 'Save and Exit' button PupilPlot store the values entered and will pass them into the Preprocess Data functions when run. Having set the preprocessing options the triggers must be defined. By clicking the Define Triggers menu option under the Preprocess Menu, the Trigger Importer window will open. All instances of ControlC will be defined as Trigger Event A, instances of GlareC will be defined as Trigger Event B, instances of GlareP will be defined as Trigger Event C, instances of ControlP will be defined as Trigger Event D. pressing the 'Show Conditions by Group' button the Trigger Importer will ask for a condition name for each group of defined triggers. As so: triggerDefinitionPP.png Clicking the 'Save Trigger Definitions and Close' button will bring us back to PupilPlot. Having set the preprocess options and defined the triggers Preprocess Data can be selected. PupilPlot will let you know if everything went according to plan and the Graph: title will reflect this as well. The other options that can be used under the Preprocess Menu can be read about on their help pages. Return to Step by Step Walk Through or PupilPlot
