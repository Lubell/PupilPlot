Mean Functional diameter by condition

This plot type option plots the mean functional difference of the pupil's diameter in millimeters. Only two conditions can be passed into this plot and the conditions can be selected using the Select Conditions option. The functions used to generate these plots are from the FDA (Functional Data Analysis) toolbox (References) and allows the diameter difference to be viewed in a much more nuanced manner in a continual manner. change in millimeter of the pupil's diameter from baseline over the course of the condition. The plotting of millimeters is a useful plot that demonstrates the physical pupil reaction to stimuli. These values have been baseline corrected are represent the mean values of all subject's pupils by condition. The Y-axis represents the percentage change from baseline and the X-axis is time. By default, all conditions will be plotted. If only a select few conditions have been selected using the Select Conditions option, then only those conditions will be plotted..
