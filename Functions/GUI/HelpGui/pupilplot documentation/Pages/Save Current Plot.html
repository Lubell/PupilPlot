Save Current Plot

The current plot displayed in PupilPlot can be saved as either a .jpeg image file or as a MATLAB figure.
  • Saving the current plot as a MATLAB figure has the advantage that the image elements can be edited at a later date.
  • Saving the current plot as .jpeg image has the advantage that it can be viewed on all types of operating systems, is smaller in size, and does not require MATLAB to view.
The user will be asked for the name they liked to give the new figure file or new jpeg file (without the file extension). PupilPlot will either make a new folder called "Figures" in the working directory and save the new file in that folder. If the folder "Figures" already exists it will save the file therein.
