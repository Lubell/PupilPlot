Load

The functions underlying this category enable you to:
  • Import new eye tracking files from Tobii and SMI eye-trackers
  • Select a directory that contains eye-tracking files that have been already imported
New Files - the menu option for importing new files. Already Imported Files - the menu option for loading files that have already been imported by PupilPlot. Upon the successful completion of either loading New Files or loading Already Imported Files, the Preprocess Menu menu will be enabled.
