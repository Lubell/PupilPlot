View Log

This menu option opens a new window that displays the current working information that PupilPlot is using. The window is adjustable and movable, but does not update in real time and will need to be closed and re-opened in order to reflect any changes that have been made since the last time it was generated. The options seen in the log are the same that are set using the Preprocess Menu menu options and are fed into the Preprocess Data functions. If using only command line functions the log can be understood as the variable that needs to be fed into the functions.
