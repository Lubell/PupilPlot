Diameters by condition without baseline removed

Sometimes it is interesting to view the data by condition without the baseline removed. The Y-axis represents the millimeter change and the X-axis is time. By default, all conditions will be plotted. If only a select few conditions have been selected using the Select Conditions option, then only those conditions will be plotted.
