Set Alpha

A significance level should be chosen prior to selecting the Run Analysis option in the Analysis Menu. The default Alpha level is set at 95% or .05. The other options are:
  • 90%
  • 95%
  • 99%
  • Other: This will open a window that will allow the user to enter their own alpha level
The results of selecting any of the alpha levels will be displayed on the command line. If an Analysis has already been run, Run Analysis will have to be chosen again in order to see the outcome of these changes. Changes to the alpha level will have an impact on the outcomes of:
  • Plot Between Subject Confidence Intervals
  • Plot Within Subject Confidence Intervals
  • Stimulus factor p-value
  • F-statistic between conditions
  • Functional t-test of difference between selected conditions
  • PCA factor loadings for selected conditions
  • Peak dilation comparison of conditions
