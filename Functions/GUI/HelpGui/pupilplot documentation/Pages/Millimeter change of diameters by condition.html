Millimeter change of diameters by condition

This Plot Type option plots the evolution in millimeters of the pupil's diameter from baseline over the course of the condition. The plotting of millimeters is a useful plot that demonstrates the physical pupil reaction to stimuli. These values have been baseline corrected are represent the mean values of all subject's pupils by condition. The Y-axis represents the percentage change from baseline and the X-axis is time. By default, all conditions will be plotted. If only a select few conditions have been selected using the Select Conditions option, then only those conditions will be plotted..
