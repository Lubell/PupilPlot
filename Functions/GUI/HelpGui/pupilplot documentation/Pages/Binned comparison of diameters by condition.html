Binned comparison of diameters by condition

This Plot Type option bins 100 milliseconds of data at key points and plots the results as a bar graph. This enables the user to see the median change in pupil diameter in millimeters at those key points. The user is asked to enter how many 100 ms bins the data should be organized into. Default is five bins. The output is an ANOVA table that can be copied and pasted into an Analysis program like SPSS, but PupilPlot just plots the results. To access the data it is recommended that the user use the pupil plot command line tools. By default, all conditions will be plotted. If only a select few conditions have been chosen using the Select Conditions option, then only those conditions will be plotted. Return to PupilPlot or Plot Type
