Already Imported Files

Use this menu option to load already imported files. PupilPlot appends _IMP to the files that it has already imported and locates them in .mat files. This means that all the files that in directory that have the _IMP.mat file extension will be loaded into PupilPlot. The status of the loading process will be printed in the MATLAB command window and the Graph: title will change to reflect this as well. After successful file loading the Preprocess Menu menu will be enabled Return to PupilPlot
