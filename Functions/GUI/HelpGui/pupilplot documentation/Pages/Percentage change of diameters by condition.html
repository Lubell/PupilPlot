Percentage change of diameters by condition

This plot type option plots the percentage change of the pupil's diameter from baseline over the course of the condition. Percentage change is a useful way to compare within condition, but generally millimeter is better for comparing across conditions and for comparing with other experimental results. These values have been baseline corrected. The Y-axis represents the percentage change from baseline and the X-axis is time. By default, all conditions will be plotted. If only a select few conditions have been selected using the Select Conditions option, then only those conditions will be plotted.
